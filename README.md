# Devsider - Propostas

Site estático em HTML/CSS.

## Publicar no Railway

1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` na raiz do projeto.
3. No Railway, escolha **Deploy from GitHub repo**.
4. Selecione o repositório.
5. Depois do deploy, vá em **Settings > Networking > Generate Domain**.

Não precisa de banco, backend, Node, React ou Docker.
